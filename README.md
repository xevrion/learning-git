## Overview
Learning Git is an essential skill for any developer, and this project aims to provide a comprehensive guide to mastering Git. This repository contains a collection of resources, including a cheatsheet and helpful tips, to aid in the learning process.

## Features
* Comprehensive cheatsheet covering Git basics and advanced topics
* Helpful tips and tricks for common Git scenarios
* Simple and easy-to-understand documentation

## Tech Stack
* Markdown for documentation
* Plain text for helpful tips and cheatsheet

## Architecture
The project consists of three main files:
* `README.md`: This file, which serves as the primary documentation for the project.
* `cheatsheet.md`: A comprehensive cheatsheet covering Git basics and advanced topics.
* `help.txt`: A plain text file containing helpful tips and tricks for common Git scenarios.

## Getting Started
### Prerequisites
* Git installed on your system
* A basic understanding of Git concepts

### Installation
To get started with this project, simply clone the repository:
```bash
git clone https://github.com/xevrion/learning-git.git
```
### Configuration
No configuration is required for this project.

## Usage
To use this project, simply navigate to the repository directory and open the `cheatsheet.md` file in your preferred Markdown viewer or editor. The `help.txt` file can be opened in any plain text editor.

### Example Usage
To view the cheatsheet, navigate to the repository directory and open `cheatsheet.md` in your preferred Markdown viewer or editor:
```bash
cd learning-git
open cheatsheet.md
```
On Linux or macOS, you can use the `xdg-open` command:
```bash
cd learning-git
xdg-open cheatsheet.md
```
On Windows, you can use the `start` command:
```bash
cd learning-git
start cheatsheet.md
```
